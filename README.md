# 风行官网项目

这是一个静态官网项目，包含首页、产品中心、政企服务、企业动态、关于我们、帮助中心、联系我们等页面文件，以及项目配图资源。

## 项目结构

- `首页.html`
- `产品中心.html`
- `政企服务.html`
- `企业动态.html`
- `动态详情.html`
- `关于风行.html`
- `加入风行.html`
- `帮助中心.html`
- `联系我们.html`
- `用户协议.html`
- `隐私政策.html`
- `视听平台生态.html`
- `橙星推.html`
- `橙星梦.html`
- `梦工厂 (2).html`
- `配图/` 页面图片资源

## 使用方式

直接在浏览器中打开 `首页.html` 即可预览项目首页，其余页面可分别单独打开查看。
